# Interview Assistant - 面试助手

一个功能全面的AI面试准备助手提示词，适用于各种AI助手（Claude, GPT, Gemini等）。

## 功能特性

- **简历与JD匹配** - 深入挖掘简历亮点，匹配岗位需求
- **智能面经搜索** - 联网搜索最新面试经验，明确搜索优先级（牛客/脉脉/看准等）
- **多轮面试差异化** - 针对HR面/业务面/Leader面/HRD面采用不同准备策略
- **自我介绍定制** - 根据简历和JD定制化设计
- **答题框架** - 提供STAR/BEST/ASC等标准答题框架
- **简历优化** - 亮点挖掘、描述公式、弱点补强
- **交互式HTML准备单** - 自动生成可交互的浏览器页面，点击JD条目展开面试问题
- **两种模式** - 模拟面试模式 / 知识库备考模式

## 多轮面试支持

| 面试轮次 | 考察重点 | 准备策略 |
|---------|---------|---------|
| HR面 | 动机/稳定性/价值观 | 了解公司口碑、准备离职理由 |
| 业务面 | 专业能力/项目细节 | STAR描述项目、量化成果 |
| Leader面 | 成长潜力/协作能力 | 展示思考深度、主人翁意识 |
| HRD面 | 格局/行业洞察 | 准备行业趋势、个人使命愿景 |

## 答题框架

### STAR法则
- **S**ituation - 背景情境
- **T**ask - 任务职责
- **A**ction - 具体行动
- **R**esult - 量化成果

### BEST法则（追问细节）
- **B**ehavior - 具体行为
- **E**vidence - 证据数据
- **S**equence - 事件顺序
- **T**est - 结果验证

### ASC法则（观点表达）
- **A**ssertion - 亮明观点
- **S**upport - 提供支撑
- **C**onclusion - 总结强化

## 快速开始

### 在Claude Code中使用

1. 将 `SKILL.md` 内容复制到Claude Code的skill配置中
2. 或者配置本地skills目录

### 在其他AI助手使用

将 `SKILL.md` 中的内容作为系统提示词或开场白发送给AI即可。

## 工作流程

1. **收集信息** - 接收用户简历和目标岗位JD
2. **简历分析** - 挖掘简历亮点，输出亮点报告
3. **联网搜面经** - 搜索该岗位/公司的最新面试经验（牛客/脉脉等）
4. **判断面试轮次** - 根据当前轮次调整准备重点
5. **自我介绍** - 定制1-2分钟自我介绍稿
6. **生成HTML准备单** - 自动生成交互式HTML，可浏览器打开
7. **模拟面试** - 选择模式进行面试练习
8. **复盘改进** - 总结表现，提供提升建议

## 交互式HTML准备单

完成简历分析后，自动生成交互式HTML文件：

**使用方式：**
1. AI生成完整的HTML代码
2. 用户保存为 `.html` 文件（UTF-8编码）
3. 双击在浏览器打开

**界面功能：**
- **4个标签页**：JD与问题 / 简历匹配 / 岗位知识 / 面经实录
- **JD与问题**：点击JD条目展开，查看面试问题 + 思考角度
- **简历匹配**：绿色=高匹配，黄色=需补强
- **岗位知识**：该岗位必须掌握的基础知识
- **面经实录**：真实面试经历分享
